# 02. DESIGN 업무 분담

담당 **14건** · 총 **83.3인일** · **43주차**까지이며, 크리티컬 패스 위의 Task는 **011**·**012**·**013**·**024**·**031**·**032**·**036** 7건으로 4명 중 가장 많다. **3~6주차의 `011 → 012 → 013`은 Phase 3 화면 Task 18건 전량의 선행**이며, 여기가 밀리면 나머지 3명이 동시에 대기한다.

## 이 문서의 위치와 규칙

- **스코프의 단일 소스는 [`ROADMAP.md`](../ROADMAP.md)다.** Task 제목·구현 사항·수락 기준이 어긋나면 ROADMAP이 이긴다
- **사람(역할·강점·가용 인일)의 단일 소스는 [`docs/team/02.DESIGN.md`](../../team/02.DESIGN.md)다**
- **전체 일정(팀 편성·크리티컬 패스·버퍼·균등 검증)의 단일 소스는 [`docs/SCHEDULE/SCHEDULE.md`](../../SCHEDULE/SCHEDULE.md)다.** 여기에 복제하지 않는다
- 이 문서는 **이 사람의 담당 Task와 그 공수·주차·선행 대기만** 적는다
- **이 디렉터리는 `docs/ROADMAP/` 아래에 있지만 `ROADMAP.md`를 고치지 않는다.** 디렉터리 위치가 소유권을 뜻하지 않는다
- **`A`/`B` 접미가 붙은 Task는 XL 분할 단위이며 ROADMAP의 Task 정의가 아니다.** 원 번호를 함께 적는다
